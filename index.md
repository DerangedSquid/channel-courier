# Channel Courier

Channel Courier is a private Discord automation bot designed to monitor voice channels, track join/leave activity, and deliver real-time audio notifications. It provides reliable, low-latency voice presence monitoring for internal servers and workflows.

---

## Features

- **Voice Channel Monitoring**  
  Detects when users join or leave voice channels in real time.

- **Audio Notifications**  
  Plays configurable audio alerts directly into the channel.

- **Webhook Event Delivery**  
  Sends structured join/leave updates to external systems.

- **Persistent Presence**  
  Maintains a stable voice connection for continuous monitoring.

- **Lightweight & Private**  
  Designed for internal use with no long-term data storage.
